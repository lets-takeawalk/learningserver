### 학습서버용 레포

- 메인 서버에 리퀘스트를 보내서 buildingInfo와 이미지URL을 받아와서 어그멘테이션 파일(연희)로 넘기는 과정
- 학습이 종료된 이후, 생성 파일을 서버와 스토리지에 올리는 함수(이후 콜백까지 고려)
- 메인서버에서 새로운 학습을 리퀘스트 받는 과정 (!!!!!!중요!!!!!)
